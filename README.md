## Project Overview

The Northern NSW Local Health District (NNSWLHD) proposes the development and deployment of an AI-Supported EMS Dispatch System to serve approximately 350,000 citizens across a 20,000 km² region.

The system addresses critical delays in the emergency treatment pathway for Acute Myocardial Infarction (AMI) by:
- Enabling citizens to contact EMS **directly**, bypassing the existing GP-intermediated pathway
- Integrating an **AI triage module** to assist dispatchers in classifying high-acuity cardiac calls
- Reducing unnecessary ambulance dispatches by a minimum of **15%**
- Delivering the solution within a **budget neutral** constraint (all costs offset by efficiency savings)

## Methodology

This project applies a **hybrid methodology** combining four frameworks:

| Framework | Role in Project |
|-----------|----------------|
| **PRINCE2** | Overarching governance — stage gates, risk management, change control, Project Board reporting |
| **Design Thinking (DT)** | User-centred analysis — empathy maps, journey maps, POV statements, HMW ideation |
| **Rational Unified Process (RUP)** | Architecture and use case modelling — Phase 2 system design |
| **Scrum** | Iterative sprint delivery — 3 × bi-weekly sprints across development phase |
