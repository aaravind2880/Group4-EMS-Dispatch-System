### 3. Cost Model Discussion
AA presented draft CAPEX structure. Key assumptions discussed and agreed:
- IS Developer hourly rate: $95/hr (NSW Health ICT Services Officer Award 2023, Grade 7)
- Project Manager hourly rate: $110/hr (NSW Health Manager State Award 2023, Level 4)
- Cost per avoided dispatch: $270 (labour + vehicle — based on 1.5hr average dispatch duration)
- Target dispatch reduction: 15% minimum (project objective)
- Avoided dispatches per year: 3,964 (calculated from 72.4 non-transport calls/day × 365 × 15%)

LH raised: should we include a contingency reserve? **Agreed: 10% of total CAPEX held as contingency.**
